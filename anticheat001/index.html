前言

哎呀 原本想讲投资理财的，可是还是觉得反诈骗比较重要，所以又来讲讲反诈骗。

事先声明，这个系列主要是教大家识破老千或诈骗集团的手法。当你了解了他们的套路后，你也可以成为他们的一分子了。 Oh不是， 开玩笑的。是可以有效的防止掉入他们的陷阱。

下面介绍几个比较经典的诈骗手法：

杀猪盘

通常都是一些口耳相传的投资项目，你在网上是找不到的。他们会以面对面或者whatsapp的方式告诉你，现在他那里有一个很hot的project，比如就是在某个国家的开发项目或者新的比特币。然后就是讲到天上有地上没有，地上没有的他那里都有，（这里我就轻描淡写的带过他们的形容词） 重点只要投资1000，等一个月后就可以拿利息100。

刚开始的一两个礼拜你真的会收到他讲的利息，之后他就会怂恿你，跟你讲1000太慢了，现在有另一个project，只要一万，等3个月就可以拿利息2000，真的是童叟无欺，如假包换的好买卖。而且这么巧，明天就是最后一天了，打铁要趁热啊，问你还在等什么？

假如你真的给了他前，时间到了你想拿回利息的时候，他就跟你讲你的投资现在赚很大，拿出来要给手续费1000. 如果你又真的天才的给了，我也跪了~

Note
  • 前面几次的投资，多数是可以拿到利润。就是要让你觉得他们可信。
  • 真正要拿钱出来的时候，就讲有手续费，不然就是说把你的钱又投资去了别的地方或者一堆借口。

破解方法

  • 给钱之前先看那个户口是私人户口还是公司户口(XXX Sdn Bhd)
  • 通常要你把钱转给个人户口的投资就是诈骗
  • 或者可以下载semak mule的App查看那个户口是不是被列入黑名单
  • 没有什么投资拿钱出来的时候会要你另外给钱 诈骗就有

诈骗电话——法庭/快递公司

就是那种在一个风和日丽的早上，突然间打电话给你，对方的声音明显是提前录制好的。 然后告诉你法庭传召你或者是你有来历不明的包裹现在在海关被扣押，要你打电话给一个特定的号码。

如果你打过去，他们就会开始讲一些威胁性的话，比如你涉嫌洗钱还是运X之类的，可以自行脑补。 最后就是要你把钱transfer 到他们的户口，不然后果很严重。我差点笑了。

Note
  • 第一次打给你的时候多数是马来文，而且就是system的声音

破解方法

  • 现在的电话都有那些CallerID 的App，建议下载，当来电者的电话号码是黑名单的时候，会提示你。这种来电就不用接了
  • 如果接了，一听到那种机械式的声音然后讲makahmah的，就可以挂了
  • 如果真的是法庭的话，是警察直接来你家。很少会打给你。
  • 如果你害怕不知道真假，直接去警察局最安全，或者联系马来西亚商业罪案调查部门里面有电话号码

爱情包裹

通常他们在社交媒体上面add你，然后跟你短信聊天， 建立男女朋友关系。他们会很主动的和你聊天，每天关心你。 这样的日子不懂过了多久，从此你们就过着幸福快乐的日子。 Oh 不是，开玩笑的，最近看太多儿童故事了。

直到有一天，他跟你讲圣诞节要到了，他已经买了一个礼物给你，就是传说中的iPhone50 Pro Max。 告诉你可能一个礼拜后你会收到。几天后，告诉你那个包裹现在有一点问题，因为是敏感货物现在卡在custom。 要你给custom的stamp duty，也就是海关税。然后你会发现，他苦口婆心的解释多多， 就是要你把钱转给一个私人/公司户口。

你转了，他就和你讲拜拜~

Note
  • 送你礼物的时间点可以是在你们认识后的一个礼拜，长的甚至可以是一年。
  • 他们通常很热心的和你聊天。
  • 选的礼物都很贵重，然后要你给10%-20%的“海关税”。

破解方法

  • 基本上，只要是照着上面的剧本走的都是诈骗
  • 他们多数会打感情牌，就是讲“Dear~ 我们都认识了那么久，我怎么可能骗你？”
  • 一旦你给了钱，老板兴兴~

社交媒体的假货

这个我个人遇过。。就是在FB上面的一些广告。那个时候我想买一个beats的头戴式耳机，market price大概是RM500。 可是看到他的广告照片很美，而且只需要RM200。点进去看到他的网址设计到很美，还有人给5星review。 也有正规的付款页面。填了地址，付了款，东西送来打开后才发现只是一个普通的耳机， 就是那种RM30就可以买到的，连那个盖住耳朵的海绵都没有！我都笑了，幸好不是给我小孩子的玩具耳机？哈哈哈

Note
  • 多数是FB上的广告
  • 产品的价钱比market的价钱便宜很多
  • 后来才发现，那些是假的，就是他们自己写的文字而已
  • 再后来，去他的FB page看，很多留言都是写被骗了，才知道是诈骗

破解方法

  • 尽量不要相信在FB上的广告贴文，特别是那种比市价便宜很多的东西
  • 多看看他的FB page下面其他post的comment，如果比较少人comment，就比较不安全
  • 不要相信他自己创建网页里面的review和comment

总结

股神巴菲特一年的平均回酬率也只是10%-15%。若有任何project超过这个值，你就要小心了，就算不是诈骗，那么他的风险也很高。
